# creative-work
YouTube Link: https://youtu.be/ezaztlhj3HM  
This is a emotion detective installation made by GSR sensor.  
The led light shows different colour when emotion value changes.  
Mode 1: Red&White  
Mode 2: Blue&Green  
Mode 3: Orange&Purple  
Mode 4:Yellow&Red
